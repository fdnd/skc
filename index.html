Over FDND

Wil jij leren hoe je interactieve toepassingen voor het web maakt? Heb jij een passie voor code, problemen oplossen en zet jij je graag in voor de eindgebruiker? En ben je op zoek naar een praktijkgerichte hbo-opleiding, maar wil je niet vier jaar lang studeren? Dan is onze nieuwe tweejarige hbo-opleiding Frontend Design & Development misschien iets voor jou!

Deze studie keuze check pagina kan jou helpen bij de keuze voor onze opleiding. Zo kun je een code challenge aangaan, om een gevoel te krijgen voor hoe het is om met HTML, CSS en JacaScript iets te ontwerpen en bouwen voor het Web. Vul het aanmeldformulier in als je interesse hebt. Als je hebt aangemeld, nodigen wij jou uit voor een intake gesprek.

Kijk hier voor meer informatie over FDND.

Code challenge

Door de code challenge te doen krijg je gevoel voor hoe het is om tew erken met de web technieken HTML, CSS en JavaScript. Tijdens de opleiding ben je continu bezig met deze technieken om interactieve toepassingen te maken voor het web.

Intake gesprek

Als je je hebt aangemeld voor FDBD nodigen wij j euit voor een intake gesprek. Tijdens de intake gaan we met je ingesprek over jou motivatie en je skills. Uit het gesprek volgt een advies over in hoeveree wij denken dat FDND een geschikte opleiding is voor jou.
